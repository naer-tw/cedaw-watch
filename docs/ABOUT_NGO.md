# 主辦組織:台灣全國媽媽護家護兒聯盟

> **建立日**:2026-04-30 / Wave 40(NGO 主辦方確認後)

## 一、本平台主辦組織

**台灣全國媽媽護家護兒聯盟**(以下簡稱「護家護兒聯盟」)為本平台之**民間主辦團體**,負責:

1. **平台對外公開發布**之具名單位
2. **預算**(公開募款 / 計畫經費)
3. **法律責任**(隱私政策、個資法、敏感資料處理)
4. **三方覆核**之第一關(主編)— 軸四旗艦議題卡(PI-21/22/23/24/25/26)發布前必經
5. **婦女顧問小組** 招募與議事(8 人,含基層 / 受暴經驗 / 原民 / 新住民 / 單親 / 高齡 / 障別)
6. **CEDAW 國際審查**(2026/2027 第 5 次審查)代表出席與影子報告提交

## 二、平台與聯盟之關係

```

```

## 四、聯盟立場與本平台立場之一致性

聯盟既有立場:**家庭核心、兒少保護、護家(護家庭、護家庭價值)**
本平台立場:**反對 CEDAW 詮釋擴張至 LGBTQ / 跨性別,主張婦女實質權益優先**

**對接關係**:
- 「**護家**」立場 → 反對非生理性別女性納入婦女家暴庇護所、女子組賽事 等女性專屬空間
- 「**護兒**」立場 → 對應 PI-26 兒童跨性別醫療議題(WHRC §9)
- **「媽媽」**身分立場 → 強化基層婦女、孕產婦、單親媽媽、原民/新住民母親 之政策優先序
- 與兒少權監督平台(同樣護家立場)生態圈互補,**不重疊**(兒少 vs 婦女)

## 四點五、與國際 gender-critical 婦運之區別與對接(預防保守派代言質疑)

護家護兒聯盟在台灣社會中之**「護家」標籤**,可能令外界將本平台與「保守宗教派」連結。本平台須明確區隔:

**本平台立場非屬保守宗教派,而與下列國際多元 gender-critical 婦運陣營對接**:

| 組織 | 意識形態 | 區隔點 |
|---|---|---|
| **WHRC**(國際)| 跨意識形態 | 全球 36,000+ 連署人含學者、律師、各派婦運 |
| **WoLF**(美國)| **左派激進女性主義** | 與宗教保守派立場明顯分野 |
| **Sex Matters**(英國)| 法律政策智庫 | 純法律論述,非宗教 |
| **Spain Contra el Borrado**| 西班牙婦運(執政左派內部分裂)| 完全非宗教 |
| **FiLiA**(英國)| 國際女性主義者大會 | 多元意識形態 |
| 北歐(Sweden / Finland)| 社會民主派 | 公衛 / 證據導向 |

**本平台立場核心**(對齊上述國際陣營,非宗教保守派):
1. **法理基礎**:CEDAW 第 1 條 sex-based 保護
2. **證據基礎**:婦女實質權益 20 年指標(GPG / TFR / 家暴 / 弱勢)
3. **婦女實質優先**:資源分流而非排他

**與宗教保守派之區別**:
- 不主張回歸傳統婚姻家庭模式
- 不反對單親 / 同性配偶之家庭(雙方為生理女性者尤其在 CEDAW 內受保護)
- 不主張禁止 LGBTQ 公民權,僅主張獨立框架處理
- 認可同婚專法(2019)現況

**與護家護兒聯盟主辦關係**:
聯盟提供平台運作之**法人載體 + 行政支援 + 媽媽社群網絡**,但本平台之**論述產出對接國際 gender-critical 婦運**,不是聯盟既有立場之延伸。聯盟「護家、護兒、媽媽身分」三大支柱與本平台立場有重疊(母性、家暴、弱勢婦女),但本平台不擔負聯盟其他政策議題(如性教育綱要、同婚評論等)。
